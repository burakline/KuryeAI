Geliştirici Notu — API Entegrasyon Mimarisi
Her platform için fetchFromPlatform(platform, apiKey) fonksiyonu tanımlı ve Firebase'e yazma altyapısı hazır.
API anahtarı geldiğinde sadece Settings → Platform Entegrasyon sekmesine yapıştırıp "Bağlantıyı Test Et" butonuna basman yeterli.
Sipariş objesi normalizer tüm platformlar için standart { customer, items, total, status, address, platform, platformOrderId } formatına çeviriyor.
